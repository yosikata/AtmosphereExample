# AtmosphereExample

UE4の4.24で使用できるAtomoSphereを使用したプロジェクト。  
テスト用に作成したプロジェクトになります。  
  
このような太陽の表現を簡単に設定を変更することで表現できます。  
![サンプル](https://github.com/yosikata/AtomosphereExample/blob/master/image.png)
